MigrantCare is a web-based platform designed to support migrant laborers by providing a centralized system for worker registration, verification, tracking, and access to essential support services. The platform aims to improve transparency, accessibility, and welfare delivery for migrant workers, especially in regions like Tamil Nadu where large migrant populations frequently face challenges in documentation, assistance, and communication.

1. Worker Registration

2. Worker Tracking

Track registered workers based on region, work site, employer, or district

Supports quick search and filtering

3. Support & Complaint System

Workers can raise complaints or request assistance

Admins can view, track, and update complaint status

Helps improve issue resolution and transparency

4. Multilingual Support

Supports multiple languages (e.g., Tamil, English, Hindi)

Ensures easy usage for workers from different states

5. Admin Dashboard

View total workers, registered employers, complaints, resolved cases, etc.

Provides real-time insights for better decision making

6. Mobile-Friendly UI

Fully responsive interface

Works smoothly on mobile devices used by workers and field volunteers
